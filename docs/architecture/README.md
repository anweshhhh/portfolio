# Architecture Snapshots

Diagram-as-code sources for the portfolio project snapshots.

These diagrams are derived from the real project repos and are intended to show a compact
container/runtime-flow view rather than a low-level implementation dump.

## Render

```bash
./scripts/render_architecture_diagrams.sh
```

That script renders the Mermaid sources in this directory to SVG assets under `assets/projects/<slug>/architecture.svg`.
